**HOW TO RUN **
1. npm install
2. npx expo start.

**REQUIREMENTS**
Download Expo Go app from playstore/appstore.
Run "npx expo start".
Press "s" in the terminal to change from development build to Expo GO
Scan QR code through camera on iOS. It will take you to webpage. Select --> Expo Go --> then --> Open App.
On Android, open Expo Go app and open camera and scan QR code.
